# Сценарии использования

Сценарии представлены по убыванию приоритета

    1. Предоставление работодателям возможности поиска кандидатур по достижениям портфолио.
        1. Выпускник модифицирует данные профиля, добавляя в своем личном кабинете достижение — новую научную публикацию.
        2. Представитель образовательного учреждения подтверждает в своем личном кабинете достижение выпускника.
        3. Работодатель выполняет расширенный поиск по профилям выпускников с учетом данных достижений.
    2. Предоставление вузам платформы для размещения портфолио учащихся и сбора расширенной аналитики для обеспечения соответствия ФГОС поколения 3++.
        1. Выпускник модифицирует данные профиля, добавляя в своем личном кабинете достижение — новую научную публикацию.
        2. Представитель образовательного учреждения подтверждает в своем личном кабинете достижение выпускника.
        3. Представитель учебного управления вуза или сотрудник Министерства науки и высшего образования получает аналитический отчет о трудоустройстве выпускников определенного года.
    3. Предоставление выпускникам возможности подбора вакансий
        1. Работодатель размещает вакансию в личном кабинете.
        2. Выпускник находит вакансию посредством инструментов поиска.
    4. Публикация отзывов работодателей о сотруднике.
        1. Работник добавляет информацию о месте работы.
        2. Работодатель, ассоциированный с данной компанией оставляет отзыв, присоединяя его к записи о месте работы .
        3. Потенциальный работодатель просматривает отзывы работодателя в записи работника.
    5. Публикация и подтверждение профессиональных компетенций и софт-навыки.
        1. Работник формирует набор профессиональных компетенций в виде списка ключевых слов через запятую.
        2. Система ассоциирует профессиональные компетенции с уже имеющимися в базе и добавляет новые.
        3. Работодатель подтверждает профессиональные компетенции работника.
        4. Зарегистрированный пользователь видит, какие профессиональные компетенции работника подтверждены.
